NLI4VolVis: Natural Language Interaction for Volume Visualization via LLM Multi-Agents and Editable 3D Gaussian Splatting

Kuangshi Ai, Kaiyuan Tang, Chaoli Wang

University of Notre Dame

IEEE VIS 2025 Best Paper Award (Top 1%)
Paper Supplementary Code arXiv
NLI4VolVis revolutionizes volume data exploration by enabling users to interact with complex 3D datasets through natural language commands via LLM multi-agents and editable 3D Gaussian Splatting, making scientific visualization more accessible and intuitive.

Abstract

Traditional volume visualization (VolVis) methods, like direct volume rendering, suffer from rigid transfer function designs and high computational costs. Although novel view synthesis approaches enhance rendering efficiency, they require additional learning effort for non-experts and lack support for semantic-level interaction. To bridge this gap, we propose NLI4VolVis, an interactive system that enables users to explore, query, and edit volumetric scenes using natural language.

NLI4VolVis integrates multi-view semantic segmentation and vision-language models to extract and understand semantic components in a scene. We introduce a multi-agent large language model architecture equipped with extensive function-calling tools to interpret user intents and execute visualization tasks.

The agents leverage external tools and declarative VolVis commands to interact with the VolVis engine powered by 3D editable Gaussians, enabling open-vocabulary object querying, real-time scene editing, best-view selection, and 2D stylization. We validate the system through case studies and a user study, highlighting improved accessibility and usability in volumetric data exploration.

System Pipeline

A multi-stage workflow integrates natural language understanding with semantic scene analysis and real-time volume visualization.

Pipeline Figure Placeholder
  1. Semantic Understanding: Multi-view semantic segmentation and vision-language models identify open-vocabulary components within the scene.
  2. Editable 3D Gaussian Rendering: The VolVis engine supports real-time scene editing, view selection, and stylization.
  3. LLM Multi-Agent Collaboration: Specialized agents decompose user requests into executable visualization commands.
  4. Natural Language Interface: Users issue plain-English requests without needing to manipulate low-level visualization parameters.

User Study

A study with eight participants examined usability, accessibility, and exploration efficiency across different experience levels.

Key Advantages

  • Improved accessibility for novice users through natural language commands.
  • Reduced cognitive load by avoiding complex parameter memorization.
  • Faster and broader exploration enabled by multi-agent task decomposition.

Identified Shortcomings

  • Ambiguous natural language occasionally required extra iterations.
  • LLM latency could interrupt the interactive flow on harder tasks.
  • Specialized domain terminology remained limited by tool coverage.
